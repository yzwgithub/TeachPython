# 字典
字典，顾名思义，与我们平时使用的字典原理非常相似，它的元素是“键-值对”的形式。
## 一、字典的定义

    a = {'a': 'apple', 'b': 'blue', 'c': 'color'}

## 二、字典的基本操作

1. 添加元素

       a = {'a': 'apple', 'b': 'blue', 'c': 'color'}
       a['d'] = 'delete'
2. 删除元素

       a = {'a': 'apple', 'b': 'blue', 'c': 'color'}
       del a['a']
3. 修改元素

       a = {'a': 'apple', 'b': 'blue', 'c': 'color'}
       a['a'] = 'append'
4. 元素索引

       a = {'a': 'apple', 'b': 'blue', 'c': 'color'}
       b = a['a']
